# 📰 NewsFlow - Додаток для перегляду новин та закладок

## 📱 Опис

**NewsFlow** - це сучасний веб-додаток для перегляду новин з можливістю створення та управління закладками. Додаток має інтуїтивний інтерфейс, адаптивний дизайн та персистентне зберігання даних.

## ✨ Основні функції

### 1. Перегляд новин 📰
- ✅ Список актуальних новин з детальною інформацією
- ✅ Красивий карткаовий дизайн кожної статті
- ✅ Зображення статей з fallback на заповнювач
- ✅ Посилання для читання повного тексту статей

### 3. Функції пошуку та фільтрації 🔍

## 📖 Гайд користувача

### Основні операції

#### Перегляд новин
1. Стартова сторінка показує список всіх новин
2. Кожна стаття містить:
   - Зображення
   - Заголовок
   - Опис
   - Джерело та дату
   - Посилання "Читати далі"
   - Кнопку для додавання закладки

#### Додавання закладки
1. Наведіть на статтю (опціонально)
2. Клацніть на кнопку **☆** (зірка) в правому кутку статті
3. Кнопка змінить колір на жовту **⭐**
4. Закладка буде збережена автоматично

#### Перегляд закладок
1. Клацніть на кнопку **"⭐ Закладки"** у верхній навігації
2. Переглядайте всі збережені статті
3. Видаліть закладку, клацнувши на **✕** або нову на **☆**

#### Пошук новин
1. Введіть текст в поле пошуку в заголовку
2. Результати оновлюються **автоматично**
3. Пошук працює по назві, описі та джерелу статей

#### Сортування
1. Виберіть опцію сортування з меню в сайдбарі
2. Опції: 
   - **"Нові першими"** - сортування за датою (спадаючи)
   - **"За назвою"** - alfabetical order (A-Z)

#### Сайдбар - статистика та останні закладки
- **Статистика** - Показує кількість всіх новин та закладок
- **Останні закладки** - Список 3 останніх закладок для швидкого доступу
- **Параметри** - Опції сортування

#### Оновлення новин
- Клацніть на іконку **🔄** у верхній навігації
- Додаток завантажує свіжий список новин
- Поточна версія використовує mock дані

## 🐛 Діагностика та розв'язання проблем

### Закладки не зберігаються
1. Перевірте, чи localStorage дозволений у браузері
   - Налаштування браузера → Конфіденційність → Дозволити локальне сховище
2. Перевірте квоту місця в браузері (зазвичай 5-10 MB)
3. Очистіть кеш браузера та перезавантажте сторінку

### Зображення не завантажуються
1. Додаток використовує Unsplash API для заповнювача
2. Перевірте підключення до інтернету
3. Зображення мають fallback на сірий заповнювач з текстом
4. Завантажте сторінку заново (Ctrl+Shift+R)

### Додаток працює повільно
1. Очистіть localStorage: `localStorage.clear()`
2. Перезавантажте сторінку (Ctrl+F5)
3. Закрийте інші вкладки браузера
4. Перезавантажте браузер

### Пошук не працює
1. Перевірте, чи JavaScript включений в браузері
2. Відкрийте консоль розробника (F12) та перевірте на помилки
3. Спробуйте в іншому браузері
4. Очистіть localStorage

## 📞 Підтримка та зворотний зв'язок

### Якщо у вас виникли проблеми:

1. **Перевірте консоль браузера** (F12) на помилки та попередження
2. **Очистіть кеш та localStorage:**
   ```javascript
   localStorage.clear()
   location.reload()
   ```
3. **Спробуйте інший браузер** для діагностики
4. **Переконайтеся**, що у вас найновіша версія браузера
5. **Перевірте підключення** до інтернету

### Консоль розробника
Для детальної діагностики див. console (F12):
```javascript
// Перевірити закладки
JSON.parse(localStorage.getItem('newsBookmarks'))

// Очистити закладки
localStorage.removeItem('newsBookmarks')

// Перевірити всі новини
console.log(window.app.allNews)
```

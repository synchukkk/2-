# 🚀 Інструкція запуску NewsFlow

## ✅ Вимоги
- Windows з встановленим Python
- Будь-який сучасний браузер (Chrome, Firefox, Edge, Safari)
- Підключення до інтернету (для завантаження зображень)

## 🎯 Способи запуску

### Спосіб 1: Подвійний клік на батник (найпростіше) ⭐
1. Знайдіть файл `run.bat` в папці додатку
2. Двічі клацніть на нього
3. Отворить автоматично **http://localhost:8000**

### Спосіб 2: Через PowerShell
1. Натисніть `Win + X` → `Windows PowerShell (Admin)` або відкрийте PowerShell
2. Виконайте:
```powershell
cd "C:\Users\yarik\OneDrive\Рабочий стол\diplomna"
python -m http.server 8000
```
3. Відкрийте браузер на http://localhost:8000

### Спосіб 3: Через командний рядок (cmd)
1. Натисніть `Win + R`, введіть `cmd`, натисніть Enter
2. Виконайте:
```cmd
cd "C:\Users\yarik\OneDrive\Рабочий стол\diplomna"
python -m http.server 8000
```
3. Відкрийте браузер на http://localhost:8000

### Спосіб 4: Прямий запуск файлу (без сервера)
1. Знайдіть файл `index.html`
2. Двічі клацніть на нього
3. Він відкриється в браузері за замовчуванням

⚠️ **Примітка**: Без локального сервера деякі функції можуть не працювати в деяких браузерах.

### Спосіб 5: У VS Code
1. Встановіть розширення "Live Server" або подібне
2. Клацніть правою кнопкою миші на `index.html`
3. Виберіть "Open with Live Server"

## 📝 Перші кроки

Після запуску додатку:

1. **Перегляд новин** - На головній сторінці ви побачите список новин
2. **Додавання закладки** - Клацніть на ⭐ (зірка) на будь-якій статті
3. **Перегляд закладок** - Натисніть "⭐ Закладки" у верхній навігації
4. **Пошук** - Введіть текст у поле пошуку для фільтрації новин

## 🔧 Налаштування

Якщо при запуску бачите помилку про Python:
- Перевірте, чи встановлений Python
- Виконайте в командному рядку: `python --version`
- Якщо не встановлений, завантажте з https://www.python.org/

Якщо порт 8000 вже використовується:
```powershell
python -m http.server 8001  # Спробуйте інший порт
```

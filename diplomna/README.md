# NewsFlow - Додаток для перегляду новин та закладок

## 📱 Опис

**NewsFlow** - це сучасний веб-додаток для перегляду новин з можливістю створення та управління закладками. Додаток має інтуїтивний інтерфейс, адаптивний дизайн та персистентне зберігання даних.

## ✨ Основні функції

### 1. **Перегляд новин**
- Список актуальних новин з детальною інформацією
- Красивий карткаовий дизайн кожної статті
- Зображення статей з fallback на заповнювач
- Посилання для читання повного тексту статей
- Дата та джерело кожної новини

### 2. **Система закладок**
- ⭐ Додавання/видалення закладок одним кліком
- Збереження закладок у localStorage
- Персистентне зберігання між сеансами
- Перегляд всіх збережених закладок
- Швидкий доступ до останніх закладок в сайдбарі

### 3. **Функції пошуку та фільтрації**
- 🔍 Пошук по назві та опису статей
- Сортування за датою або назвою
- Реалізація debounce для оптимізації пошуку
- Миттєві результати фільтрації

### 4. **Інтерфейс**
- Липкий заголовок з навігацією
- Адаптивний дизайн для всіх пристроїв
- Темні та світлі області для кращої читабельності
- Плавні анімації та переходи
- Статистика новин та закладок

## 🛠️ Технічний стек

## 🚀 Запуск додатку

### Варіант 1: Простий запуск
1. Відкрийте файл `index.html` в браузері
2. Клацніть на файл двічі або потягніть його в браузер

### Варіант 2: Локальний сервер (рекомендовано)

**Python (встановлений за замовчуванням на Windows)**
```bash
# Python 3.9+
python -m http.server 8000

# Python 2.7
python -m SimpleHTTPServer 8000
```
Потім відкрийте http://localhost:8000 в браузері

**Node.js (якщо встановлений)**
```bash
npx http-server
```

**Live Server в VS Code**
- Встановіть розширення "Live Server"
- Клацніть правою кнопкою миші на `index.html`
- Виберіть "Open with Live Server"

## 📋 Гайд користувача

### Основні операції

#### Перегляд новин
1. Стартова сторінка показує список всіх новин
2. Кожна стаття містить:
   - Зображення
   - Заголовок
   - Опис
   - Джерело та дату
   - Посилання "Читати далі"
   - Кнопку для додавання закладки

#### Додавання закладки
1. Наведіть на статтю (опціонально)
2. Клацніть на кнопку ☆ (зірка) в правому кутку статті
3. Кнопка змінить колір на жовту ⭐

#### Перегляд закладок
1. Клацніть на кнопку "⭐ Закладки" у верхній навігації
2. Переглядайте всі збережені статті
3. Видаліть закладку, клацнувши на ✕ або нову на ☆

#### Пошук
1. Введіть текст в поле пошуку
2. Результати оновлюються автоматично
3. Пошук працює по назві та описі статей

#### Сортування
1. Виберіть опцію сортування з меню
2. Опції: "Нові першими" або "За назвою"

## 💾 Збереження даних

- **localStorage** - закладки зберігаються в браузері
- Ключ: `newsBookmarks`
- Формат: JSON масив статей
- Автоматичне збереження при кожній зміні

### Експорт закладок
```javascript
// В консолі браузера
console.log(JSON.parse(localStorage.getItem('newsBookmarks')))
```

### Видалення всіх закладок
```javascript
// В консолі браузера
localStorage.removeItem('newsBookmarks')
```

## 🐛 Діагностика

### Закладки не зберігаються
1. Перевірте, чи localStorage дозволений у браузері
2. Перевірте квоту місця в браузері
3. Очистіть кеш браузера

### Зображення не завантажуються
- Додаток використовує Unsplash API
- Перевірте підключення до інтернету
- Зображення мають fallback на заповнювач

### Додаток працює повільно
- Очистіть localStorage
- Перезавантажте сторінку
- Закрийте інші вкладки

## 👨‍💻 Розробка

### Основні класи та методи

**NewsApp** - головний клас додатку
- `init()` - ініціалізація додатку
- `loadNews()` - завантаження новин
- `toggleBookmark()` - додавання/видалення закладки
- `render()` - рендеринг інтерфейсу
- `filterAndSort()` - фільтрація та сортування

### Розширення функціоналу

Для додавання нових функцій:
1. Додайте новий метод до класу `NewsApp`
2. Оновіть `render()` для показу змін
3. Додайте відповідний слухач подій в `attachEventListeners()`
